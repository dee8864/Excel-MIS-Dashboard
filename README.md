# Excel-MIS-Dashboard
Excel MIS Dashboard — Call Center Break & Efficiency Tracker
🚀 Excited to share my latest Excel project — an interactive MIS Dashboard built on real-world call center data!

I worked on a dataset of 149+ call center agents across 3 cities (Mumbai, Hyderabad & Delhi), containing detailed records spread across multiple sheets — and transformed it into a fully interactive, insight-driven dashboard.

📊 Here's what the project covers:
🔹 Sheet 1 — Break Analysis
🔹 Sheet 2 — Efficiency Metrics
🔹 Sheet 3 — Overall Call Performance
🔹 Sheet 4 — Consolidated Data
🔹 Dashboard (=DASHBOARD)

A fully interactive Excel dashboard featuring:
✅ PivotTables for dynamic data summarization
✅ PivotCharts — Bar chart (Working Hours) + Pie chart (Active With Customer %)
✅ Slicers for interactive employee-level filtering
✅ Dual Report view for side-by-side KPI comparison
✅ Clean layout with navigation controls

💡 Key skills demonstrated:
→ Advanced Excel (PivotTables, PivotCharts, Slicers, Dashboard Design)
→ Data Cleaning & Consolidation across multi-sheet workbooks
→ KPI Design & Performance Metrics for Call Center MIS
→ Time-format data handling (HH:MM:SS)
→ Data Storytelling & Visual Reporting
→ Multi-location workforce data management
→ Record Macro ( VBA )

This project helped me strengthen my ability to turn raw, complex datasets into clean, decision-ready dashboards — a critical skill in any data or operations role.

Always learning. Always building. 📈
